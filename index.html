Lab Submissions

Lab 1: UI Design

Focus on layout structure, alignment, and basic composables in Jetpack Compose.

This lab helped me understand the basics of user interface design in mobile applications. I learned how layout structure, colors, and alignment affect user experience. It improved my awareness of designing clean and user-friendly screens.

Lab 2: Interaction

Handling state, button clicks, and dynamic user inputs.

In this lab, I learned how user interactions such as button clicks and input handling work. It helped me understand how apps respond dynamically to user actions. This made my application feel more interactive and functional.

Lab 3: Material Design

Implementing Material 3 components for a polished visual identity.

This lab introduced me to Material Design principles in Android development. I learned how to use components like cards, buttons, and consistent spacing. It helped improve the visual quality of my app.

Lab 4: Navigation

Setting up a NavHost to manage transitions between multiple screens.

In this lab, I learned how to move between different screens using navigation components. It made my app more structured and easier to use. I also understood how navigation improves user flow.

Lab 5: Room Database

Persistent local storage using SQL-based Room library.

This lab taught me how to store data locally using Room database. I learned how to save, update, and retrieve data inside the app. It helped me understand offline data management in Android.
